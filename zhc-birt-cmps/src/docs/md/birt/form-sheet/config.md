### Attributes 属性

| 名称        | 说明          | 类型                | 可选值 |    默认值     |
|-------------|---------------|---------------------|--------|:-------------:|
| controlName | 类名          | string              | -      | BirtFormSheet |
| controlId   | 控件ID,唯一值 | string              | -      |       -       |
| style       | 自定义样式    | string              | -      |       -       |
| width       | 宽度          | int,string,function | -      |       -       |
| height      | 高度          | int,string,function | -      |       -       |
| children    | 子控件列表    | array               | -      |       -       |
