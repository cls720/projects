### Attributes 属性

| 名称         | 说明                   | 类型      | 可选值 |       默认值       |
|--------------|------------------------|-----------|--------|:------------------:|
| controlName  | 类名                   | string    | -      | WebSocketConection |
| controlId    | 控件ID,唯一值          | string    | -      |         -          |
| webSocketUrl | WebSocket连接路径      | string    | -      |         -          |
| target       | 消息推送目标数据集列表 | json      | -      |         -          |
| webSocket    | WebSocket连接实例对象  | WebSocket | -      |         -          |
